Ici sont créés l'apparence et le contenu principal des pages du site dans les header/footer.
Par exemple le menu, l'entête du site, le bas de page...

Pour cela suivre l'exemple du mainheader et du mainfooter.

Le dossier packagecss détermine un thème de css complet à choisir pour votre site.
Il suffit de paramétrer le thème choisi dans le fichier conf/conf.design.php
