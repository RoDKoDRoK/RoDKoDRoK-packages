Pour créer une page ajax, consulter la page ajax "example" pour indication.

D'abord choisir un nom : test
Créer un template test.tpl dans le dossier template.
Créer un fichier test.php dans le dossier flux.
Créer une classe class.test.php dans le dossier class/fluxajaxclass (optionnel mais récommandé).

Créer un fichier ajax_test.css dans le dossier css/pagecss (optionnel).
Créer un fichier ajax_test.js dans le dossier js/pagejs (optionnel).

Créer un fichier ajax_test_fr_fr.xml, ajax_test_en_en.xml, ... dans le dossier /lang à la racine (optionnel).

Des classes d'outils pratiques sont disponibles dans le dossier /core/class/pratikclass que vous pouvez inclure et appeler dans votre code.
