Pour créer un webserice, consulter le ws example pour indication.

D'abord choisir un nom : test
Créer un template test.tpl dans le dossier template.
Créer un fichier test.php dans le dossier flux.
Créer une classe class.test.php dans le dossier class/fluxwsclass (optionnel mais récommandé).

Créer un fichier ws_test_fr_fr.xml, ws_test_en_en.xml, ... dans le dossier /lang à la racine (optionnel).

Des classes d'outils pratiques sont disponibles dans le dossier class/pratikclass que vous pouvez inclure et appeler dans votre code.
Ces classes peuvent être appelées à l'aide de la fonction $includer->include_pratikclass("pratikclassname")
