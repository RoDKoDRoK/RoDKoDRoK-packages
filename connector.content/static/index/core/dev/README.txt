Pour créer une page, consulter la page example pour indication.

D'abord choisir un nom : test
Créer un template test.tpl dans le dossier template.
Créer un fichier test.php dans le dossier page.
Créer une classe class.test.php dans le dossier class/pageclass (optionnel mais récommandé).

Créer un fichier test.css dans le dossier css/pagecss (optionnel).
Créer un fichier js dans le dossier js/pagejs (optionnel).

Créer un fichier test_fr_fr.xml, test_en_en.xml, ... dans le dossier /lang à la racine (optionnel).

Des classes d'outils pratiques sont disponibles dans le dossier class/pratikclass que vous pouvez inclure et appeler dans votre code.
