Pour mettre à jour une librairie, télécharger la librairie dans le dossier integrate/lib.
Puis créer un fichier integrate/libintegrator/libintegrator.<type_librairie>.<nom_librairie>.php pour charger la lib dans le site. Reprendre le format des autres fichiers.
<type_librairie> peut être ajax, tpl, wysiwyg ou other selon la fonction de votre librairie extérieure.
Certaines librairies nécessitent un driver pour son utilisation dans le RODFramework à placer dans le dossier integrate/driver.
