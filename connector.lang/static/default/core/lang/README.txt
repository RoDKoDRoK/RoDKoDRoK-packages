Ici se trouvent les fichiers de traduction du site et des différentes pages du site.
Pour la langue fr_fr par exemple, vous trouverez :
 - un fichier fr_fr.xml pour tout le site
 - un fichier <nompage>_fr_fr.xml pour une page en particulier
 - un fichier ajax<nomfluxajax>_fr_fr.xml pour un flux ajax en particulier

Il est possible de créer des fichiers pour autant de langues souhaitées.
